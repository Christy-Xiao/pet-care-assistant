# 养宠助手 (Pet Care Assistant)

一个基于AI的智能养宠助手Web应用，帮助宠主管理宠物健康、追踪护理日程、查找周边宠物友好场所。

## 核心功能

### P0 - 必须实现
- [x] 宠物健康档案系统 - 记录品种、过敏史、健康日记
- [x] 视觉初筛模块 - 照片分析与健康问题识别
- [x] 草坪搜索引擎 - 基于LBS检索周边宠物友好绿地
- [x] 护理日程管理 - 疫苗、驱虫到期提醒

### P1 - 应做实现
- [x] 天气出行建议 - 结合气象数据推荐出行时段
- [x] 多宠切换功能 - 支持管理多只宠物

## 技术栈

- **框架**: Next.js 14 (App Router)
- **样式**: Tailwind CSS
- **状态管理**: React Context + LocalStorage
- **地图**: 高德地图 JS API
- **AI**: 多模态分析（模拟）

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
```

## 功能说明

### 1. 宠物档案
- 添加/编辑宠物信息（品种、年龄、体重、过敏史）
- 记录健康日记和就诊历史
- 查看历史记录时间线

### 2. 健康分析
- 上传宠物照片（粪便、皮肤等）
- AI模拟分析并给出建议
- 根据宠物档案提供个性化建议

### 3. 绿地搜索
- 显示周边3km内宠物友好绿地
- 按草坪面积和人流筛选
- 一键导航到目的地

### 4. 护理日程
- 自动生成疫苗、驱虫日程
- 到期前3天自动提醒
- 记录完成状态
